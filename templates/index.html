Bem-vindo ao StockSys - Sistema de Estoque

O StockSys é um sistema web de estoque de produtos, utilizando Python com Flask e MySQL

Com este sistema, o usuário pode cadastrar produtos, consultar, editar e excluir. Verificar quantidade no Estoque valores de compra e valroes para a revenda.

Este projeto foi desenvolvido como parte de um dentro do Python com o Flask.
